# 나의 뜨개 프로젝트 기록

뜨개 프로젝트, 실, 도안, 바늘, 부자재를 한 곳에서 기록하고 관리하는 개인용 웹앱입니다.

## 주요 기능
- 프로젝트 상태 관리: 예정 / 진행중 / 완성 / 보류
- 실 재고 관리
- 도안 / 바늘 / 부자재 라이브러리
- 프로젝트와 자재 연결
- 사진 업로드
- JSON 백업 / 복원
- 로컬 저장 기반 사용

## 왜 만들었나요?
뜨개를 하다 보면
- 어떤 실이 얼마나 남았는지
- 어떤 도안으로 무엇을 떴는지
- 바늘과 부자재를 무엇을 썼는지
한 번에 정리하고 싶어집니다.

이 앱은 그런 기록을 한 곳에 모으기 위해 만들었습니다.

## 기술 스택
- React
- TypeScript
- Vite
- Tailwind CSS
- Dexie (IndexedDB)
- React Router
- Vitest
